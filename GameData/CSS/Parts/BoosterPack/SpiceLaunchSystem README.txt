SPICE LAUNCH SYSTEM 0.1.2
By Maccollo

The purpose of this mod is to provide a set of parts to create an accurate representation of the SLS rocket in combination with realism overhual (RO. The mod is updated as more parts are created.

INSTALLTION
Place the "Spice Launch System" in the gamedata directory.

FEATURES
ATK 5 segment booster: 
A 5 segment version of the STS booster. 
Core oxygen and hydrogen tanks, intertank and interstage.
